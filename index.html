Proyecto integrado con Arquitectura de Software

Este dashboard hace parte de un proyecto trabajado en conjunto con la asignatura de Arquitectura de Software, tomando como referencia la plataforma Spotify.

Referencia: Spotify utiliza microservicios, APIs y procesamiento de grandes volúmenes de datos para analizar el comportamiento de los usuarios.
Aplicación en el proyecto: se integran playlists del grupo y un dataset global de Spotify para realizar el análisis musical.
Propósito: transformar datos musicales en patrones, perfiles y conocimiento útil para responder las preguntas del dashboard.
